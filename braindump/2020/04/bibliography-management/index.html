Bibliography Management

What do the files mean?

Here, I store all bibliography entries in article.bib , pdfs to pdfs folder, notes in articles.org file. That’s all there is to understand.

  (setq bibtex-completion-library-path "c:/Users/sagar/Dropbox/pdfs/"
      bibtex-completion-bibliography "c:/Users/sagar/Dropbox/org/references/articles.bib"
      bibtex-completion-notes-path "c:/Users/sagar/Dropbox/org/references/articles.org"
      bibtex-completion-notes-path "c:/Users/sagar/Dropbox/org/references/articles.org")

  (setq org-ref-notes-directory "c:/Users/sagar/Dropbox/org/notes/"
      org-ref-bibliography-notes "c:/Users/sagar/Dropbox/org/references/articles.org"
      org-ref-default-bibliography '("c:/Users/sagar/Dropbox/org/references/articles.bib")
      org-ref-pdf-directory "c:/Users/sagar/Dropbox/pdfs")

Using these files

  1. Adding entries:
    • Find a paper, add it to aricles.bib. Either DOI or paper link.
    • If pdf found by org-ref, okay.
    • Else doi-utils-get-bibtex-entry-pdf run it.
    • Now, on reading make an entry to articles.org file.
      • Either using org-ref-open-bibtex-notes
      • If article.bib not open, then use helm-bibtex.
  2. Taking notes:
    • org-noter for this
    • Searching: helm-bibtex.
