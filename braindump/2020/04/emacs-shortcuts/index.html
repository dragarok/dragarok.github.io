Emacs Shortcuts

Emacs Shortcuts

  1. Spc f d –> search for files in dired
  2. Spc : –> main command
  3. Spc g s –> git status
  4. Spc n n –> Capture template
  5. Spc n a –> agenda view
  6. Vim commands
    1. Ctrl up , down –> go up and down in file
    2. q [a-z] –> record a macro
    3. Shift » –> indent in code
  7. Evil snipe:
    1. s ch : go to ch ; same as f but more letters
    2. gc : to comment using evil
    3. gs [annotation] –> to go to lines directly like vimium ease
    4. dii –> delete inside indentation
    5. dik –> delete inside plus a line above
