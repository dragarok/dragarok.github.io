CTF New attacks and tools:

Hash extender attack: https://github.com/iagox86/hash%5Fextender

Decrypt substitution cipher using guesses: https://quipqiup.com/

Xor cipher: python3 -m pip install –user xortool

examples: xortool file.bin xortool -l 11 -c 20 file.bin xortool -x -c ' ' file.hex xortool -b -f -l 23 -t base64 message.enc

The most common use is to pass just the encrypted file and the most frequent character (usually 00 for binaries and 20 for text files) - length will be automatically chosen:

Hydra:

hydra -l root -P /usr/share/wordlists/metasploit/unix_passwords.txt -t 6 ssh://192.168.1.123 : 6 threads, login as root, use wordlist unix_passwords.txt

Pw-inspector:

pw-inspector -i /usr/share/wordlists/nmap.lst -o /root/passes.txt -m 6 -M 10 : get passes of specific length from wordlist

John jumbo : new package of john the ripper

Nozzlr : bruteforce utility, manages threads, multiple wordlists,

ophcrack: modular utility to crack using rainbow tables in windows

patator: crack various types of login such as ftp login, smtp, ssh, telnet login and so on

one_gadget: gem install one_gadget When playing ctf pwn challenges we usually need the one-gadget RCE (remote code execution), which leads to call execve('/bin/sh’, NULL, NULL).

This tool automates it , no need to ida and stuff

e.g. one_gadget /lib/x86_64-linux-gnu/libc.so.6

qira: qemu based runtime analyzer tool

zmap: open source security scanner
