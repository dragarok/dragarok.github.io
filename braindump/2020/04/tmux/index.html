Tmux

mastering tmux:

tmux new -s “session_name”

tmux ls - lists sessions

Pane controls:

everything is with control + B

then:

" : split horizontally

% : split vertically

o : go to next pane

; : previous pane

q : show pane numbers

z : toggle pane zoom ( can hide all panes using z)

! : convert pane into an window

Control + O : swap panes

t: copy mode

{: move to previous pane

Space: toggle between pane layouts

c: create new window

d: detach from the session

,: rename current windows

&: close a window

w: list window

p: previous window

n: next window

After getting to copy mode:

prefix + enter to copy text

p to paste

j, k, l, h to navigate

g and G to get to top and bottom

Alt + . to get the last word i typed

ctrl + e to get to the end of line

ctrl + a to get to the beginning of the line

can bind tmux prefix key to cntrl + a or something like that to nest tmux sessions
