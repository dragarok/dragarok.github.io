Git Commands

–tags: Useful

Commands

  • git add : tracking with git

  • git status : current status of tracked, untracked, changed, committed files.

  • git add filename.extension

(only after making the file it can be done)

  • double add command - stages the new changes

  • git commit

  • git add . -> all files are committed

  • git add *.html ->only html files are committed

  • touch .gitignore

  • git branch Mybranch

  • git commit -m ‘commit command’

  • git checkout master

  • git checkout Mybranch

  • git stash -> saves everything over a branch that can be applied later to it

  • come back and work continously into it: git stash apply

Working with remote repos:

  • git remote

  • git clone website

  • cd location

  • git remote -v

  • git fetch origin

  • git pull origin: will pull from the repo and also merge it

  • git remote add Myrepo website

Advanced

  • git rm –cached `git ls-files -i –exclude-from=.gitignore`: remove cached files that you have now added to .gitignore .
    • e.g. config files you added initially.

Clearing branch history or certain commits

  • git checkout –orphan cleanup_branch_name
  • git add .
  • git commit -am “First commit”
  • git cherry-pick commit-name to commit-name : which commits you want to stay. If you want everything gone then no need to cherry pick
  • git branch -D master
  • git branch -m master : renames cleanup_branch_name to master.
  • git push origin master
    • if it doesn’t work
    • git push -f origin master
