Emacs Shortcuts

Emacs Shortcuts

  1. Spc f d –> search for files in dired
  2. Spc : –> main command
  3. Spc g s –> git status
  4. Spc n n –> Capture template
  5. Spc n a –> agenda view
  6. Vim commands
    1. Ctrl up , down –> go up and down in file
    2. q [a-z] –> record a macro
    3. Shift » –> indent in code
  7. Evil snipe:
    1. s ch : go to ch ; same as f but more letters
    2. gc : to comment using evil
    3. gs [annotation] –> to go to lines directly like vimium ease
    4. dii –> delete inside indentation
    5. dik –> delete inside plus a line above
  8. Dired buffer shortcuts:
    1. Dired marking and unmarking options: % and * keys
    2. Narrow subtree : (
    3. Show git status for files : )
    4. Diff one file to another: =
    5. Create a directory : +
    6. Up a directory : -
    7. Rename a file: R
    8. Change timestamp of file: T
    9. Copy filename: Y
    10. Open a sub-directory in same menu: I
    11. Change ownership of file: O
    12. Unmark all files: U
    13. Hardlink of file: H
    14. Goto file: J
    15. Counsel search online: K
    16. Encrypt, decrypt a file: ;
    17. Uncompress or compress : Z
    18. Copy file to : C
    19. Byte compile a file: B
    20. Change file mode: M
    21. Copy async to rsync : C-c C-r
    22. Wide dired mode: i
  9. Org table by copying html table: C-c |
  10. Show all project TODOs: Spc p t
  11. Scroll other window : S-up
  12. Scroll other window down : S-down
