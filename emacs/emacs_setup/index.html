Epdfinfo libpoppler error

Newer libpoppler won’t be identified by epdfinfo. So, link a new one to act as old one and it works all good.

sudo ln -s libpoppler.so.99 libpoppler.so.98
sudo ln -s libpoppler.so.99.0.0 libpoppler.so.98.0.0

Elfeed setup

The configuration for elfeed is inside the config file. Nonetheless, the primary work to be done in this case is that I need to know how to put new rss into my elfeed file.     <comment>Emacs Org-mode File</comment>
    <glob pattern="*.org"/>
    <alias type="text/org"/>
  </mime-type>
</mime-info>

Org Protocol

xdg-mime default org-protocol.desktop x-scheme-handler/org-protocol

In .local/share/applications/org-protocol.desktop, add the following:

[Desktop Entry]
Name=Org-Protocol
Exec=emacsclient %u
Icon=emacs-icon
Type=Application
Terminal=false
MimeType=x-scheme-handler/org-protocol
