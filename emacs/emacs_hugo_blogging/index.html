    Truly minimal as well as well off with content. He is using bootstrap and hugo to build this site and has a resource to tell how to build site and theme with bootstrap.

    • Creates a new site and a new theme as well

      1. hugo new site mysite
      2. cd mysite
      3. hugo new theme mytheme

    • Go to proj dir and edit config.toml

      1. theme=“mytheme”
      2. relativeURLs=“true” —-Relative content using dirs and subdirs to manage and use the theme we just created.

    • hugo server

      hugo server –watch –verbose –buildDrafts –cleanDestinationDir –disableFastRender

      • disabling fast render reloads and gets latest version of file on shift reload.

    • Now make theme proper

      • themes/mytheme/layouts/_default/baseof.html
