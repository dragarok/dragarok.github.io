  • Smerge and Magit to resolve git conflicts:

    • Use magit to merge or any cli tool.
      • It will tell manage merge conflicts and will stop you.
    • Go to magit see unmerged changes section.
    • Then open from magit the conflicted files and will show you head and other branch output that are in conflict with each other.
    • C-c ^ a ==> a means accept all changes : it keeps both
    • C-c ^ u ==> upper change
    • C-c ^ l ==> lower change
    • C-c ^ n ==> go to next conflict
    • C-c ^ p ==> go to previous conflict
