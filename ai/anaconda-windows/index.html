Anaconda Installation in Windows

  1. Issue with pyzmq:

       pip uninstall pyzmq
   pip install pyzmq

  2. Issue with requests

       conda deactivate
   conda update --force conda

  3. Issue with sqlite3

    • Find sqlite 3 dll
    • https://www.sqlite.org/download.html
    • Put it into Anaconda3/DLLS/ folder

  4. Install Python 3.6 or 3.7 if you want same env for PyTorch as well as for working with TensorFlow-GPU.

  5. Anaconda if HTTPS error:

       conda --config ssl_verify=false

  6. Path variable has importance of order. Based on which python you want as default, put that higher in PATH variable list in windows.

  7. To install notebook:

    1. just do pip install notebook

  8. conda list command : same as pip freeze : shows who installed it conda or pip as well as the versions.
